# Expense Tracker CLI

A command-line app built in Python to track daily expenses by category.

## Concepts Used
- OOP with Encapsulation (private attributes, getters)
- JSON file handling (persistent storage)
- Exception Handling (invalid input, missing file, empty file)
- Modular code structure (3 separate files)

## How to Run
python main.py

## Features
- Add expenses with title, amount and category
- View all expenses
- View total amount spent
- Filter expenses by category (case-insensitive)
- Data persists between sessions
